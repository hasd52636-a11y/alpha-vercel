# 客户端部署说明文档

## 项目简介

**阿尔法01** 是一个基于OpenClaw和smart008-2026的智能客服系统，使用智谱AI作为后端大脑，提供多模态交互能力。本文档详细介绍客户端的安装和配置步骤。

## 客户端环境要求

### 硬件要求
- CPU: 至少 Intel Core i3 或同等性能
- 内存: 至少 4GB RAM
- 存储: 至少 50GB 可用空间
- 网络: 稳定的互联网连接
- 摄像头: 用于视频聊天功能（可选）
- 麦克风: 用于语音交互功能（可选）

### 软件要求
- 操作系统: Windows 10/11、macOS 10.15+、Linux (Ubuntu 20.04+)
- Node.js: v24.x 或更高版本（仅开发模式需要）
- npm: v10.x 或更高版本（仅开发模式需要）

## 部署方式

### 方式一：桌面应用安装包（推荐）

#### Windows 安装

1. **下载安装包**
   - 从官方网站下载 `Alpha01-Setup.exe` 文件
   - 或从部署包中获取 `dist-electron` 目录下的安装文件

2. **安装应用**
   - 双击 `Alpha01-Setup.exe` 文件
   - 按照安装向导的提示完成安装
   - 选择安装目录（默认: `C:\Program Files\Alpha01`）

3. **首次运行**
   - 从桌面快捷方式启动应用
   - 进入初始设置向导

#### macOS 安装

1. **下载安装包**
   - 从官方网站下载 `Alpha01.dmg` 文件

2. **安装应用**
   - 双击 `Alpha01.dmg` 文件
   - 将 `Alpha01` 拖拽到 `Applications` 文件夹

3. **首次运行**
   - 从 Launchpad 启动应用
   - 进入初始设置向导

#### Linux 安装

1. **下载安装包**
   - 从官方网站下载对应发行版的安装包（.deb、.rpm 或 AppImage）

2. **安装应用**
   - **Debian/Ubuntu**: `sudo dpkg -i alpha01_1.0.0_amd64.deb`
   - **CentOS/RHEL**: `sudo rpm -ivh alpha01-1.0.0-1.x86_64.rpm`
   - **AppImage**: `chmod +x Alpha01-1.0.0.AppImage && ./Alpha01-1.0.0.AppImage`

3. **首次运行**
   - 从应用菜单启动应用
   - 进入初始设置向导

### 方式二：开发模式运行

#### 1. 环境准备

1. **安装 Node.js**
   ```bash
   # Windows
   # 从 https://nodejs.org/ 下载并安装 LTS 版本
   
   # macOS/Linux
   curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
   source ~/.bashrc
   nvm install 24
   nvm use 24
   ```

2. **克隆代码**
   - 将 `client-deployment` 目录复制到本地

3. **安装依赖**
   ```bash
   cd client-deployment
   npm install
   ```

#### 2. 运行应用

1. **开发模式**
   ```bash
   npm run dev
   ```
   - 应用会在浏览器中打开（http://localhost:3000）

2. **Electron 开发模式**
   ```bash
   npm run electron:dev
   ```
   - 应用会在 Electron 窗口中打开

3. **构建应用**
   ```bash
   # 构建 Web 版本
   npm run build
   
   # 构建桌面应用
   npm run electron:build
   ```

## 初始设置向导

### 1. 服务器配置

- **服务器地址**: 默认为 `https://sora2.wboke.com`
- **端口**: 默认为 `443`
- **验证连接**: 点击 "测试连接" 按钮验证服务器连接

### 2. API 密钥配置

- **选择 AI 提供商**: 智谱AI
- **输入 API 密钥**: 从智谱AI官网获取的 API 密钥
- **验证密钥**: 点击 "验证密钥" 按钮验证 API 密钥有效性

### 3. 企业信息配置

- **企业名称**: 输入您的企业名称
- **联系人**: 输入联系人姓名
- **联系电话**: 输入联系电话
- **企业 logo**: 上传企业 logo（可选）

### 4. 产品配置

- **添加产品**: 点击 "添加产品" 按钮
- **产品名称**: 输入产品名称
- **产品描述**: 输入产品描述
- **产品知识库**: 上传或创建产品知识库

## 功能配置

### 1. 智能客服设置

- **响应模式**: 选择 "快速响应" 或 "详细解答"
- **语言设置**: 选择支持的语言
- **工作时间**: 设置客服工作时间
- **自动回复**: 配置自动回复消息

### 2. 多模态功能

- **语音交互**: 启用/禁用语音交互
- **视频聊天**: 启用/禁用视频聊天
- **图片分析**: 启用/禁用图片分析
- **文件上传**: 配置允许上传的文件类型和大小

### 3. 知识库管理

- **导入知识**: 支持 Excel、CSV、PDF 格式
- **分类管理**: 创建和管理知识分类
- **搜索设置**: 配置搜索算法和权重
- **更新频率**: 设置知识库自动更新频率

### 4. 技能库管理

- **添加技能**: 点击 "添加技能" 按钮
- **导入技能包**: 上传技能包文件（JSON 格式）
- **技能配置**: 配置技能参数和触发条件
- **技能启用/禁用**: 切换技能状态

## 使用说明

### 1. 商家后台

- **访问地址**: `https://sora2.wboke.com/admin/dashboard`
- **登录方式**: 首次使用默认无需登录，后续可在设置中开启身份验证
- **主要功能**:
  - 控制面板: 查看系统状态和统计信息
  - 产品管理: 管理产品信息和二维码
  - 数据分析: 查看交互数据和分析报告
  - 工单管理: 处理用户工单
  - API 设置: 配置 AI 模型和 API 密钥
  - 知识库: 管理产品知识库
  - 知识图谱: 查看知识关联和结构

### 2. 二维码生成

1. **进入产品管理**
   - 点击左侧菜单 "产品管理"

2. **选择产品**
   - 选择要生成二维码的产品

3. **生成二维码**
   - 点击 "生成二维码" 按钮
   - 选择二维码样式和大小
   - 点击 "下载二维码" 按钮保存图片

4. **二维码管理**
   - 查看已生成的二维码列表
   - 刷新、禁用或删除二维码

### 3. 客服交互

1. **接收用户消息**
   - 用户扫描二维码后，消息会显示在 "工单管理" 页面

2. **回复用户**
   - 点击消息进入对话界面
   - 输入回复内容
   - 点击 "发送" 按钮

3. **使用 AI 助手**
   - 在对话界面中，点击 "AI 助手" 按钮
   - 选择 AI 响应模式
   - 点击 "生成回复" 按钮
   - 编辑 AI 生成的回复
   - 点击 "发送" 按钮

4. **多模态交互**
   - **语音回复**: 点击麦克风按钮录制语音
   - **视频通话**: 点击视频按钮发起视频通话
   - **图片分析**: 上传图片进行分析
   - **文件发送**: 点击文件按钮发送文件

## 故障排除

### 常见问题

1. **应用无法启动**
   - 检查系统版本是否满足要求
   - 检查网络连接是否正常
   - 查看应用日志文件

2. **服务器连接失败**
   - 检查服务器地址是否正确
   - 检查网络连接是否正常
   - 检查服务器是否正在运行

3. **API 密钥验证失败**
   - 检查 API 密钥是否正确
   - 检查 API 密钥是否过期
   - 检查网络连接是否正常

4. **二维码无法扫描**
   - 检查二维码是否清晰
   - 检查二维码是否过期
   - 检查服务器连接是否正常

5. **AI 响应缓慢**
   - 检查网络连接速度
   - 检查服务器负载
   - 调整 AI 模型参数

### 日志文件位置

- **Windows**: `%APPDATA%\Alpha01\logs`
- **macOS**: `~/Library/Logs/Alpha01`
- **Linux**: `~/.config/Alpha01/logs`

### 紧急情况处理

1. **重置应用**
   - 进入 "设置" > "高级"
   - 点击 "重置应用"
   - 确认重置操作

2. **恢复备份**
   - 进入 "设置" > "备份与恢复"
   - 选择备份文件
   - 点击 "恢复" 按钮

3. **联系支持**
   - 进入 "帮助" > "联系支持"
   - 填写支持请求
   - 附上应用日志
